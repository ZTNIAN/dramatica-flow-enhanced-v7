# 写作技巧库

## 一、开篇钩子技巧

### 1. 时间反常钩子
```
凌晨三点十七分，我又醒了。
不是因为噩梦——我已经很久不做梦了——
是被一种声音吵醒的。
```
**要点**：具体时间 + 反常现象

### 2. 空间反常钩子
```
这门，我明明锁了三道。
现在，它开着。
```
**要点**：安全空间的异常

### 3. 人物反常钩子
```
"你不是死了么。"
这不是问句。
```
**要点**：应该死的人出现了

### 4. 常用开头句式
- "雨。冷雨。冷雨如刀..."
- "太久了。久到..."
- "这个时间，不该有人敲门。"
- "我明明锁了门。现在，它开着。"
- "那双眼，太静了。"

## 二、五感描写模板

### 雨夜场景
- **视觉**：路灯昏黄、雨丝斜织、地面反光
- **听觉**：雨声淅沥、雷声轰鸣、水滴滴答
- **嗅觉**：泥土腥气、潮湿霉味
- **触觉**：冷风扑面、雨水冰凉
- **心理**：压抑、孤独、期待

### 恐怖场景
- **视觉**：阴影晃动、光线闪烁、不明物体
- **听觉**：脚步声、呼吸声、吱呀声
- **嗅觉**：血腥味、腐臭味
- **触觉**：手心出汗、后背发凉
- **心理**：恐惧、猜疑、逃避

### 密室场景
- **视觉**：蜡烛摇曳、阴影晃动、尘埃浮动
- **听觉**：自己的呼吸、心跳、远处滴水
- **嗅觉**：霉味、铁锈味、陈年灰尘
- **触觉**：墙壁冰冷、地面潮湿

### 战场场景
- **视觉**：血色、烟尘、残破旗帜
- **听觉**：喊杀声、兵器碰撞、惨叫
- **嗅觉**：血腥味、焦糊味
- **触觉**：伤口疼痛、盔甲沉重

## 三、人物出场模板

### 神秘人物
```
她穿着一件黑色的雨衣，帽子压得很低。
但[主角]还是一眼就认出了她。

不是因为脸。
是因为她右手上的那道疤。
```
**要点**：不直接描写外貌，用特征暗示身份

### 反派出场
```
他笑起来的时候，眼睛不笑。
那双眼，太静了。
静得像口枯井。
```
**要点**：反常的细节（笑眼不笑）

## 四、对话技巧

### 1. 潜台词示例
```
"你变了。"
"哪里？"
"以前你会先问，是谁的血。"

（潜台词：你现在不关心真相了）
```

### 2. 打断技巧
```
"大爷，这方圆十里——"
"满了，去别处吧。"

（打断制造紧张感）
```

### 3. 省略技巧
```
"因为——"她忽然停住了。

（省略比说出更有悬念）
```

### 4. 答非所问
```
"你到底在怕什么？"
"今天的茶不错。"

（回避本身就是答案）
```

## 五、节奏控制技巧

### 紧张节奏
```
短句。
短句。
短句。
像心跳。
```
**要点**：10字以内短句，动作密集，无描写

### 舒缓节奏
```
长句像河流一样流淌，
带着读者慢慢进入场景，
感受氛围的每一个细节。
```
**要点**：允许20字以上长句，感官细节丰富

### 快慢交替
```
紧张！紧张！紧张！（快）

然后突然放慢——（慢）
让读者喘口气，
再接着紧张！
```
**要点**：3-5段紧张后1-2段舒缓（张弛交替）

## 六、章末钩子类型

### 1. 信息钩子
```
"他是你儿子。"
```

### 2. 悬念钩子
```
"三日后，勿开门。"
```

### 3. 选择钩子
```
"这是答案。或者是另一个陷阱。你自己选。"
```

### 4. 画面钩子
```
街角的路灯下，似乎有一个小小的身影一闪而过。
那是个孩子。
穿着蓝色的雨衣，戴着一顶红色的帽子。
```

### 5. 结尾余韵技巧
```
（事实陈述）
他只知道，从这一刻起，

（情感升华）
他平静了三年的生活，
彻底地，
永远地，
结束了。

（疑问/悬念）
而那个叫XX的孩子——
如果是真的——
那他这三年来，
到底错过了什么？
```

## 七、Show Don't Tell 转换表

| Tell（禁止） | Show（推荐） |
|-------------|-------------|
| 他很生气 | 他捏碎了茶杯/声音发抖/摔门而去 |
| 她很害怕 | 她往后退了半步/手心出汗/不敢看 |
| 他很累 | 他瘫在椅子上/连手指都不想动 |
| 她很伤心 | 筷子在碗里搅了十分钟一口没吃 |
| 他很紧张 | 在原地转圈/深呼吸/手指敲桌子 |
| 气氛很紧张 | 没人说话只有钟表声/有人清了清嗓子 |
| 他很尴尬 | 手不知道往哪放/假装看窗外 |
| 她吃醋了 | 筷子夹了个空/话突然少了 |

**核心原则**：用动作、物件、价格说话，禁止直接说"感到XX"

## 八、情绪升级阶梯

### 愤怒
不悦 → 不满 → 恼怒 → 愤怒 → 暴怒 → 冷静的愤怒

### 恐惧
不安 → 紧张 → 害怕 → 恐惧 → 惊恐 → 麻木

### 悲伤
失落 → 难过 → 悲伤 → 绝望 → 平静

### 爱慕
好奇 → 注意 → 心动 → 忐忑 → 坦然 → 执着

**每次升级需要触发事件，不能平白无故升级。**

## 九、伏笔技巧

### 埋设手法
- **道具伏笔**：一件不起眼的东西，后面揭示关键信息
- **对话伏笔**：角色无意中说的话，后面兑现
- **行为伏笔**：角色反常的行为，后面解释原因
- **环境伏笔**：场景中的细节，后面揭示含义

### 回收原则
- 必须有新信息
- 应该产生后果
- 读者看到时"啊原来如此"

### 铺垫距离
- 短伏笔：1-3章内回收
- 中伏笔：5-15章内回收
- 长伏笔：20-50章内回收
- 超过50章未回收 → 预警

## 十、快速修复表

| 问题 | 快速修复 |
|------|---------|
| "首先" | 改为"一开始"或直接删除 |
| "感到很XX" | 改为动作，如"手抖"、"心跳快" |
| 句子太长 | 切成2-3个短句 |
| 细节不够 | 加颜色/声音/气味 |
| 太像AI | 加1个错别字或口语重复 |
| 对话太规整 | 加动作打断/答非所问 |
| 信息堆砌 | 分散到行动中展示 |
