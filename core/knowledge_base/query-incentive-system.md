# 知识库查询激励系统

## 查询奖励机制

### 基础奖励
| 查询行为 | 积分奖励 | 说明 |
|---------|---------|------|
| **首次查询** | +2分 | 每次任务首次查询知识库 |
| **多次查询** | +1分/次 | 同一任务多次查询（上限3次） |
| **正确查询** | +3分 | 查询后正确应用规则 |
| **主动查询** | +5分 | 未提示情况下主动查询 |

### 查询质量奖励
| 质量等级 | 奖励 | 标准 |
|---------|------|------|
| **优秀** | +5分 | 查询精准，一次找到所需信息 |
| **良好** | +3分 | 查询2-3次找到所需信息 |
| **合格** | +1分 | 查询多次但最终找到 |
| **低效** | -2分 | 反复查询同一内容或未应用 |

### 累计奖励
| 累计查询次数 | 称号 | 额外奖励 | 特权 |
|-------------|------|---------|------|
| 10次 | 知识学徒 | +5分 | 无 |
| 50次 | 知识探索者 | +20分 | 优先获取新规则通知 |
| 100次 | 知识专家 | +50分 | 可推荐新增知识库内容 |
| 500次 | 知识大师 | +200分 | 拥有知识库评审权限 |

## 查询日志要求

每次查询必须在回复中记录：

```markdown
【知识库查询记录】
- 查询时间：2026-03-18 11:00
- 查询文件：kb:rules/de-ai-guidelines
- 查询原因：不确定"首先"是否算AI词汇
- 应用情况：已确认"首先"属于禁止词，已删除
- 查询效果：精准找到答案
```

## 必须查询的场景（不查询扣分）

以下场景**必须查询**知识库，否则扣5分：

- [ ] **不确定95分标准具体是什么** → 查询 kb:rules/review-criteria-95
- [ ] **不确定某表达是否AI味** → 查询 kb:rules/de-ai-guidelines
- [ ] **被返工后需要改进** → 查询 kb:examples/before-after/
- [ ] **需要某类写作技巧** → 查询 kb:references/writing-techniques/
- [ ] **不确定红线清单** → 查询 kb:rules/redlines

## 查询激励提醒

**每次任务开始时，系统会提醒**：
```
💡 提示：查询知识库可获得积分奖励！
   首次查询 +2分，正确应用 +3分，主动查询 +5分
   累计查询还可获得称号和特权！
```
