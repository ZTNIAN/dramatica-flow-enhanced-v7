# 开篇与结尾设计指南（OpeningEndingDesigner 方法论）

## 黄金三章设计（Opening Design）

### 第一章：开篇钩子（Hook the Reader）
**目标**：前3秒抓住读者，300字内建立好奇心

#### 开篇类型
1. **悬念开篇**：从一个谜团/异常事件开始
   - 「他醒来时，发现自己躺在一口棺材里。」
2. **动作开篇**：从一个紧张的行动场景开始
   - 「子弹从他耳边擦过，留下一道血痕。」
3. **对话开篇**：从一句引人入胜的对白开始
   - 「'你以为你是谁？'女孩冷笑着说。」
4. **反常开篇**：从一个违反直觉的场景开始
   - 「整个城市的人都在逃跑，只有他一个人在往里走。」

#### 开篇禁忌
- 禁止从「平凡的早晨」开始（起床/刷牙/出门）
- 禁止大段世界观说明（先用剧情吸引人）
- 禁止大段人物外貌描写（读者不关心还没认识的角色）
- 禁止以「在很久很久以前」模式开场

### 第二章：人物引入（Establish Character）
**目标**：让读者认识主角，建立情感连接

#### 引入要素
1. **即时性格展示**：通过行动而非描述来展示性格
2. **处境揭示**：主角当前面临的处境/困境
3. **目标暗示**：暗示主角的渴望/目标
4. **缺陷暗示**：暗示主角性格中的缺陷/弱点
5. **能力展示**：展示主角的独特能力（可以不完全展示）

#### 引入技巧
- Show don't tell：用行动展示性格，不用「他是一个XX的人」
- 先同情后好奇：先让读者同情主角的处境，再激发好奇心
- 三个场景：至少用3个不同场景展示主角的不同侧面

### 第三章：世界观引入（Establish World）+ 悬念预埋
**目标**：自然地引入世界观，埋下长线伏笔

#### 世界观引入原则
- **融入剧情**：通过角色的日常行动自然展示世界规则
- **分批引入**：不要一章说清所有世界观
- **先用后解**：先展示现象，后解释原因

#### 悬念预埋
- 在前三章中埋下至少3个伏笔
- 一个短期伏笔（3-5章内解决）
- 一个中期伏笔（10-20章内解决）
- 一个长期伏笔（贯穿全书）

## 全书结尾设计（Ending Design）

### 高潮对决（Climax）
- 主角面临最大的挑战/选择
- 所有铺垫在此汇聚
- 必须展示主角的成长/改变

### 主题升华（Theme Resolution）
- 回应开头提出的主题
- 通过事件和选择来升华，而非说教
- 读者应该自己领悟主题

### 情感落点（Emotional Landing）
- 给读者一个情感上的满足感
- 可以是喜/悲/五味杂陈
- 要与全书的基调一致

### 尾声设计（Epilogue）
- 给每个主要角色一个交代
- 不要所有角色都完美结局（不真实）
- 留一点想象空间给读者
