# 小说作家专属技能库

## 一、开篇钩子技巧

### 1. 时间反常钩子
```
凌晨三点十七分，我又醒了。
不是因为噩梦——我已经很久不做梦了——
是被一种声音吵醒的。
```
**要点**：具体时间 + 反常现象

### 2. 空间反常钩子
```
这门，我明明锁了三道。
现在，它开着。
```
**要点**：安全空间的异常

### 3. 人物反常钩子
```
"你不是死了么。"
这不是问句。
```
**要点**：应该死的人出现了

## 二、五感描写模板

### 雨夜场景
- **视觉**：路灯昏黄、雨丝斜织、地面反光
- **听觉**：雨声淅沥、雷声轰鸣、水滴滴答
- **嗅觉**：泥土腥气、潮湿霉味
- **触觉**：冷风扑面、雨水冰凉
- **心理**：压抑、孤独、期待

### 恐怖场景
- **视觉**：阴影晃动、光线闪烁、不明物体
- **听觉**：脚步声、呼吸声、吱呀声
- **嗅觉**：血腥味、腐臭味
- **触觉**：手心出汗、后背发凉
- **心理**：恐惧、猜疑、逃避

## 三、人物出场模板

### 神秘人物
```
她穿着一件黑色的雨衣，帽子压得很低。
但[主角]还是一眼就认出了她。

不是因为脸。
是因为她右手上的那道疤。
```
**要点**：不直接描写外貌，用特征暗示身份

### 反派出场
```
他笑起来的时候，眼睛不笑。
那双眼，太静了。
静得像口枯井。
```
**要点**：反常的细节（笑眼不笑）

## 四、对话技巧

### 1. 潜台词示例
```
"你变了。"
"哪里？"
"以前你会先问，是谁的血。"

（潜台词：你现在不关心真相了）
```

### 2. 打断技巧
```
"大爷，这方圆十里——"
"满了，去别处吧。"

（打断制造紧张感）
```

### 3. 省略技巧
```
"因为——"她忽然停住了。

（省略比说出更有悬念）
```

## 五、节奏控制技巧

### 紧张节奏
```
短句。
短句。
短句。
像心跳。
```

### 舒缓节奏
```
长句像河流一样流淌，
带着读者慢慢进入场景，
感受氛围的每一个细节。
```

### 快慢交替
```
紧张！紧张！紧张！（快）

然后突然放慢——（慢）
让读者喘口气，
再接着紧张！
```

## 六、章末钩子类型

### 1. 信息钩子
```
"他是你儿子。"
```

### 2. 悬念钩子
```
"三日后，勿开门。"
```

### 3. 选择钩子
```
"这是答案。或者是另一个陷阱。你自己选。"
```

### 4. 画面钩子
```
街角的路灯下，似乎有一个小小的身影一闪而过。
那是个孩子。
穿着蓝色的雨衣，戴着一顶红色的帽子。
```

## 七、去AI味自检清单

每写500字检查：
- [ ] 删除所有"首先/其次/最后"
- [ ] 删除"更关键的是/有意思的是"
- [ ] 检查"感到XX"，改为动作展示
- [ ] 增加至少1处具体细节
- [ ] 增加至少1种新感官
- [ ] 短句是否占60%以上
- [ ] 有没有1-2处故意的小瑕疵

---

## 八、快速修复表

| 问题 | 快速修复 |
|------|---------|
| "首先" | 改为"一开始"或直接删除 |
| "感到很XX" | 改为动作，如"手抖"、"心跳快" |
| 句子太长 | 切成2-3个短句 |
| 细节不够 | 加颜色/声音/气味 |
| 太像AI | 加1个错别字或口语重复 |

## 十、结尾余韵技巧

```
（事实陈述）
他只知道，从这一刻起，

（情感升华）
他平静了三年的生活，
彻底地，
永远地，
结束了。

（疑问/悬念）
而那个叫XX的孩子——
如果是真的——
那他这三年来，
到底错过了什么？
```
