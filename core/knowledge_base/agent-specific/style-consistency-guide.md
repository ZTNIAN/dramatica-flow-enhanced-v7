# 风格一致性检查指南（StyleConsistencyChecker 方法论）

## 五维一致性检查

### 1. 文笔风格一致性
- **语言风格**：全书语言风格是否统一（典雅/通俗/简洁/华丽）
- **描写密度**：每千字的描写占比是否一致
- **修辞使用**：比喻/排比/拟人等修辞手法的使用频率
- **句式偏好**：短句为主还是长句为主，比例是否稳定

### 2. 人物语气一致性
- **词汇习惯**：角色的用词风格是否前后一致
- **句式特征**：角色的说话句式是否变化一致
- **语气基调**：角色的主要语气是否稳定
- **口头禅**：角色的口头禅是否一致出现

### 3. 叙事节奏一致性
- **章节长度**：各章字数是否在合理范围内波动
- **场景切换**：场景切换的频率和方式是否一致
- **张弛交替**：紧张和舒缓的交替节奏是否稳定
- **对话比例**：对话和叙述的比例是否合理

### 4. 时代背景一致性
- **用语风格**：语言风格是否符合时代背景
- **物品描写**：提到的物品是否符合时代
- **社会制度**：描写的社会制度是否一致
- **风俗习惯**：角色的行为是否符合时代

### 5. 情感基调一致性
- **整体基调**：全书的情感基调是否稳定（热血/暗黑/治愈/轻松）
- **情感波动**：情感波动的幅度是否合理
- **情感层次**：情感表达是否有多层次
- **读者体验**：读者的情感体验是否连贯

## 检查流程

### 跨章节对比
- 随机抽取3-5个章节进行对比
- 检查同一维度在不同章节的表现
- 记录不一致的具体表现

### 偏差判定标准
- **轻微偏差**：不影响阅读体验，可忽略
- **中度偏差**：影响阅读体验但不严重，建议修改
- **严重偏差**：严重影响阅读体验，必须修改

### 报告格式
- 维度名称
- 偏差程度（轻微/中度/严重）
- 具体表现（引用原文）
- 修改建议
