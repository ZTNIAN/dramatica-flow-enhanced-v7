# 角色成长指南（CharacterGrowthExpert 方法论）

## 八维度角色构建

### 1. 基础设定
- 姓名（含字号/绰号）
- 外貌（身高/体型/特征/穿着风格）
- 年龄
- 身份/职业

### 2. 性格设定
- 核心性格（1-3个关键词）
- 性格成因（为什么是这样）
- 性格缺陷（完美角色不讨喜）
- 性格反差（外表vs内在的反差）
- 性格锁定（绝对不做的事）

### 3. 家世与经历
- 家庭背景
- 成长经历（关键节点）
- 人生转折点
- 人际关系网络

### 4. 喜好与习惯
- 喜欢什么/讨厌什么
- 日常习惯/小动作
- 口头禅/习惯用语
- 饮食/作息偏好

### 5. 能力设定
- 战斗技能
- 非战斗特长
- 能力成长空间
- 能力上限和代价

### 6. 成长轨迹
- 初期（1-30%）：迷茫/受挫/被压制
- 中期（30-60%）：觉醒/成长/突破
- 后期（60-90%）：蜕变/担当/面对
- 终局（90-100%）：完成/超越/牺牲

### 7. 关键转折点（至少3个）
- 认知转折：角色对世界的看法改变
- 能力转折：角色的能力发生质变
- 情感转折：角色的情感关系发生改变
- 价值观转折：角色的价值观发生改变

### 8. 人物关系矩阵
- 与每个主要角色的关系类型
- 关系强度（-100 到 +100）
- 关系发展预期
- 关系中的关键事件
