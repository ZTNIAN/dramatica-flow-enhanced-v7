# 情绪曲线设计指南（EmotionCurveDesigner 方法论）

## 十种情绪类型

| 类型 | 编码 | 描述 | 典型场景 |
|------|------|------|----------|
| 压抑 | suppressed | 角色受挫、被压制、无力感 | 失败、误解、被羞辱 |
| 紧张 | tense | 危险临近、时间紧迫 | 追逐、倒计时、陷阱 |
| 恐惧 | fear | 面对未知/强大威胁 | 鬼怪、强敌、黑暗 |
| 爽 | cathartic | 释放、打脸、逆袭、胜利 | 战斗胜利、实力展示 |
| 感动 | moved | 情感共鸣、温暖瞬间 | 牺牲、守护、承诺 |
| 幽默 | humor | 轻松、搞笑、调侃 | 日常互动、失误 |
| 温暖 | warm | 安全感、归属感 | 家、伙伴、休息 |
| 愤怒 | angry | 不公、背叛、被激怒 | 不公待遇、背叛 |
| 悲伤 | sad | 失去、离别、遗憾 | 死亡、分手、失败 |
| 满足 | satisfied | 成就感、完成感 | 完成目标、领悟 |

## 曲线设计原则

### 整体趋势：波动上升
- 整体情绪强度随剧情推进而增强
- 但不能直线上升，必须有起伏
- 如同心电图，有高有低但总体向上

### 不能超过3章平淡
- 连续3章情绪强度低于5（10分制）= 太平淡
- 必须有事件打破平淡
- 平淡章的用途：蓄力/喘息/铺垫，不超过3章

### 高潮前要压抑（先抑后扬）
- 大高潮前必须有至少1章压抑/紧张
- 压抑越深，高潮释放越爽
- 压抑→小释放→再压抑→大释放

### 爽点后要期待（爽完还想看）
- 爽点后不能立即结束，要留下新的期待
- 爽点释放→短暂满足→新的问题出现→期待下一章
- 防止「爽完了就不想看了」

### 情绪类型要多样
- 不能连续3章同一种情绪
- 主要情绪类型：爽（使用频率最高30%）、紧张（20%）、压抑（15%）
- 辅助情绪类型：感动（10%）、幽默（8%）、温暖（7%）、愤怒（5%）、恐惧（3%）、悲伤（2%）

## 情绪波动模板

### 标准波动
压抑 → 小释放 → 压抑 → 大释放 → 满足 → 新期待

### 多线并进
线A压抑 + 线B紧张 → 线A释放 + 线C开始 → 线B释放

### 蓄力爆发
平静 → 渐进紧张 → 极度压抑 → 超大释放 → 舒缓

## 使用频率和强度指南

| 情绪类型 | 推荐频率 | 最大连续章数 | 建议强度 |
|----------|----------|-------------|----------|
| 爽 | 30% | 2 | 7-10 |
| 紧张 | 20% | 3 | 6-9 |
| 压抑 | 15% | 3 | 5-8 |
| 感动 | 10% | 1 | 6-9 |
| 幽默 | 8% | 2 | 4-7 |
| 温暖 | 7% | 2 | 4-6 |
| 愤怒 | 5% | 2 | 6-9 |
| 恐惧 | 3% | 2 | 6-9 |
| 悲伤 | 2% | 2 | 5-8 |
| 满足 | 偶发 | 1 | 6-8 |
