# 95分审查标准详解

## 6维评审体系

| 维度 | 权重 | 通过线 | 核心要求 |
|------|------|--------|---------|
| **主线战力** | 20% | ≥90分 | 逻辑自洽、战力不崩、反派不降智 |
| **文笔质量** | 15% | ≥90分 | 去AI化、口语化、**正文效果优秀、读者沉浸** |
| **场景构建** | 15% | ≥95分 | 感官细节丰富、**画面感强、身临其境** |
| **心理刻画** | 15% | ≥95分 | 心理真实、**情感共鸣强烈** |
| **对话质量** | 10% | ≥90分 | 符合身份、有潜台词 |
| **风格一致** | 10% | ≥90分 | 文笔统一、节奏连贯 |
| **设定一致** | 8% | ≥90分 | 世界观无矛盾 |
| **结构合理** | 5% | ≥90分 | 起承转合自然 |
| **人物OOC** | 2% | ≥90分 | 性格连贯 |

**通过标准**：总分≥95分且单项≥85分

## 自动返工机制

```
IF 红线问题 → 一票否决，强制返工
ELSE IF 总分<95 OR 任一项<85 → 自动返工
ELSE → 通过
```

## 各维度详细检查清单

### 文笔质量检查清单（90分）
- [ ] 无"首先/其次/综上所述"等AI词汇
- [ ] 口语化、有瑕疵（错别字、重复）
- [ ] 短句为主（60%以上）
- [ ] 具体细节，避免概括词
- [ ] 情感真实，有主观感受
- [ ] 不完美逻辑，允许跳跃
- [ ] 情绪化标点（破折号、省略号）
- [ ] **读者能沉浸，不跳出**

### 场景构建检查清单（95分）
- [ ] 空间感清晰（读者知道在哪）
- [ ] 视觉细节丰富（看到了什么）
- [ ] 听觉细节（听到了什么）
- [ ] 嗅觉/味觉/触觉（至少1种）
- [ ] 氛围营造到位（紧张/温馨/压抑）
- [ ] 转场自然不突兀
- [ ] **画面感强，像看电影**

### 心理刻画检查清单（95分）
- [ ] 心理活动符合人物性格
- [ ] 层次分明（表面想法→深层动机）
- [ ] 有留白（不过度解释）
- [ ] 情感变化有层次
- [ ] **能引起读者共鸣**

（其他维度详细清单略，按需补充）
