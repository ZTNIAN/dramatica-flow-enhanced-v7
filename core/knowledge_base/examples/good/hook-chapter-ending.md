# 悬念钩子示例

## 类型1：信息钩子

**示例**：
```
"他是你儿子。"
```

**效果**：
- 颠覆主角认知
- 引出更多问题（为什么是儿子？什么时候的事？）

**适用**：
- 章末
- 真相揭露时刻

---

## 类型2：选择钩子

**示例**：
```
"这是答案。或者是另一个陷阱。你自己选。"
```

**效果**：
- 主角面临两难
- 读者想知道怎么选

**适用**：
- 章末
- 关键决策点

---

## 类型3：警告钩子

**示例**：
```
"三日后，勿开门。"
```

**效果**：
- 明确的警告
- 时间限制（三日后）
- 具体行动（勿开门）

**适用**：
- 章末
- 预示危险

---

## 类型4：画面钩子

**示例**：
```
街角的路灯下，似乎有一个小小的身影一闪而过。

那是个孩子。

穿着蓝色的雨衣，戴着一顶红色的帽子。

他只看到了一眼，然后那个身影就消失在雨中了。

但那一眼，足够了。

那双眼睛，和他一模一样。
```

**效果**：
- 具体画面（孩子、蓝雨衣、红帽子）
- 反常（眼睛一模一样）
- 留下想象空间

**适用**：
- 章末
- 暗示联系

---

## 类型5：疑问钩子

**示例**：
```
他不知道那是谁。

他不知道发生了什么。

他只知道，从这一刻起，他平静了三年的生活，彻底地，永远地，结束了。

而那个叫周小鱼的孩子——

如果是真的——

那他这三年来，到底错过了什么？
```

**效果**：
- 多个疑问
- 情感升华
- 引导下一章

**适用**：
- 章末
- 第一卷/第一部分结尾

---

## 钩子设计原则

### 1. 简洁
- 不要大段解释
- 一句话或一个画面

### 2. 具体
- 不要笼统
- 要有细节（"蓝色的雨衣"比"一个人"好）

### 3. 反常
- 正常→反常
- 预期→反转

### 4. 留白
- 不要说满
- 让读者想象

---

## 钩子位置

### 章末钩子
- 引导读下一章
- 常用：信息钩子、警告钩子

### 卷末钩子
- 引导读下一卷
- 常用：疑问钩子、重大转折

### 情节钩子
- 推动剧情
- 常用：选择钩子

---

## 练习

为下面场景设计钩子：

**场景**：主角打开一个盒子，发现里面有一张照片。

**钩子示例**：
```
照片里，是一个女人和一个孩子。

女人他认识。三年前死的。

孩子——

他看着孩子的眼睛，那双眼和他的一模一样。

照片背面有一行字，墨迹已经晕开了，但还是能看清：

"你欠我们的。"
```
