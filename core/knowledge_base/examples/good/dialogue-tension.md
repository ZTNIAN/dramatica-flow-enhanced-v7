# 紧张对话示例

## 示例1：逼问

```
"谁派你来的？"

女人没有回答。她的眼睛看向窗外，像是在数雨点。

"我在问你话。"他的声音低下去，不是生气，是那种——更让人害怕的——平静。

"你知道我不会说的。"

"我知道你会。"他向前走了一步，"每个人都有一个价。你的价是什么？"

女人终于转过头来，看着他。那眼神很奇怪，不是恐惧，不是愤怒，是——怜悯？

"你付不起的。"
```

**技巧分析**：
- 用动作打断对话（"眼睛看向窗外"）
- 不说"他很生气"，而说"声音低下去"
- 用破折号制造停顿和悬念

---

## 示例2：试探

```
"你认识老周？"

"哪个老周？"她反问得太快了。

他笑了笑，没说话。那个笑容很淡，像水面上的一层涟漪，转瞬即逝。

"山下客栈的那个。"他说，"三年前死的那个。"

她的手指动了一下，很小的一个动作，但他注意到了。

"不认识。"

"真的？"他从口袋里掏出一张照片，放在桌上，"可他认识你。"

照片里，她站在客栈门口，笑得很开心。日期显示：三年前。
```

**技巧分析**：
- 用细节暗示（"手指动了一下"）
- 不说谎，但隐瞒（"反问得太快了"）
- 抛出证据，制造压力

## 示例3：摊牌

```
"你知道多少？"

"足够让你睡不着。"

他攥紧了拳头，但没有挥出去。他还不能动她，至少现在不能。

"你想要什么？"

"真相。"她说，"三年前那天晚上，到底发生了什么。"

"那对你不重要。"

"那对我很重要。"她从怀里掏出一样东西，扔在桌上，"因为我妹妹死在那晚。"
```

**技巧分析**：
- 简短有力的对话
- 最后抛出重磅信息
- 用动作展示克制（"攥紧了拳头，但没有挥出去"）

---

## 技巧总结

### 1. 不说满
```
❌ "我知道你是凶手！"
✅ "你知道多少？"（试探，不说满）
```

### 2. 用动作打断
```
❌ A说，B说，A说...（枯燥）
✅ A说，B看窗外，A等，B转过头说...（有层次）
```

### 3. 潜台词
```
表面："你想要什么？"
潜台词："我怕你了，我要收买你"
```

### 4. 最后一句最重要
把最重要的信息放在对话的最后一句，让读者印象深刻。
