# 神秘人物出场示例

## 类型1：从特征暗示

```
她穿着一件黑色的雨衣，帽子压得很低。

但陈默还是一眼就认出了她。

不是因为脸。

是因为她右手上的那道疤。

从虎口一直延伸到手腕，像一条粉白色的蜈蚣。那是三年前，她替他挡那一刀时留下的。
```

**技巧**：
- 不直接说"他认出了她"
- 用一个特征（疤）暗示身份
- 这个特征还要有故事（替他挡刀）

---

## 类型2：从反常行为

```
那姑娘站在雨里，斗篷湿漉漉地贴在身上。

看起来也就十七八岁的样子，眉眼清秀，就是眼神有点怪——

太静了。

静得像口枯井，看不出半点波澜。

这个年纪的姑娘，不该有这种眼神。
```

**技巧**：
- 先正常描写（十七八岁、眉眼清秀）
- 然后反常（眼神太静）
- 评论（"不该有这种眼神"）

---

## 类型3：从他人反应

```
老张头从门缝里往外看，只能瞧见一个模糊的影子。

"满了，去别处吧。"

"大爷，"那声音更轻了，"这方圆十里，就您这一户人家。"

老张头犹豫了一下。

的确，最近的村子也在二十里开外...

等等，她怎么知道？
```

**技巧**：
- 通过对话展示
- 透露信息（她知道"方圆十里"）
- 主角意识到反常

---

## 类型4：延迟揭示

```
门外站着一个人。

一个女人。

她穿着一件黑色的雨衣...

（500字后）

"你不是死了么。"

这不是问句。

女人抬起头。

雨水顺着帽檐流下来...
```

**技巧**：
- 先描述，不揭示身份
- 延迟到关键时刻
- 用对话或动作揭示

---

## 类型5：从氛围营造

```
那声音很轻，像是怕惊着什么似的，三下，又三下。

老张头愣了一下。这天气，这时辰，哪来的客人？

他抄起门边的柴刀...

"谁啊？"

"过路的，求宿一晚。"

是个女人的声音，听起来很年轻...

但老张头注意到，她的手指太干净了。

不像是在雨里走了二十里的人该有的手指。
```

**技巧**：
- 先营造神秘氛围
- 声音、动作引导
- 最后用一个细节揭穿（手指太干净）

---

## 技巧总结

### 1. 不要一上来就说
❌ "一个神秘的女人出现了"
✅ 先描写，让读者自己觉得神秘

### 2. 用一个特征暗示
- 疤痕、眼神、动作
- 这个特征要有意义

### 3. 反常之处
- 正常中的反常
- 让人物觉得"不对劲"

### 4. 延迟揭示
- 不要立即揭示身份
- 保持悬念

### 5. 与环境互动
- 不是孤立存在
- 与场景、他人互动

---

## 避免错误

❌ **不要**：
- "一个神秘的人出现了"
- "他看起来很神秘"
- 大段外貌描写

✅ **要**：
- 用细节暗示
- 让他人反应
- 反常之处
