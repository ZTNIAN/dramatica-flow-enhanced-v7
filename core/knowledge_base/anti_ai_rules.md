# 去AI味规则库

## 绝对禁止词（出现即扣分）
首先/其次/最后/总之/综上所述/众所周知/不言而喻/毫无疑问/显而易见
更关键的是/更奇怪的是/更有意思的是/值得注意的是
在这个XX的时代/让我们来看看/让我们一起
一方面/另一方面/不仅...而且...

## 高频AI词（每3000字各最多1次）
仿佛/忽然/竟然/不禁/宛如/猛地/顿时
深深/缓缓/轻轻/默默
眼神中闪过一丝/嘴角微微上扬
心中涌起一股/一股XX涌上心头
空气突然安静/气氛突然凝固

## 套话模板（禁止使用）
XX的尽头是XX / XX不过是XX的另一种形式 / 人生就像XX
这个世界上有两种人 / 所有的XX都是XX

## 句式雷区

### 报告式语言（禁止）
从客观角度来看 / 综合各方面因素考虑 / 这充分说明了 / 分析了形势后

### 元叙事（禁止）
核心动机是 / 叙事节奏 / 人物弧线 / 这个情节的设计是为了

### 集体反应套话（禁止）
全场震惊 / 众人哗然 / 所有人都愣住了 / 在场的人无不 / 无一例外地

## Show Don't Tell 转换表
| Tell（禁止） | Show（推荐） |
|-------------|-------------|
| 他很生气 | 他捏碎了茶杯/声音发抖/摔门而去 |
| 她很害怕 | 她往后退了半步/手心出汗/不敢看 |
| 他很累 | 他瘫在椅子上/连手指都不想动 |
| 她很伤心 | 筷子在碗里搅了十分钟一口没吃 |
| 他很紧张 | 在原地转圈/深呼吸/手指敲桌子 |
| 气氛很紧张 | 没人说话只有钟表声/有人清了清嗓子 |
